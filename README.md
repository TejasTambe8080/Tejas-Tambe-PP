# Tejas Portfolio (React + Vite)

Professional portfolio built with React, Vite, Tailwind CSS, and Framer Motion.

## Local Development

1. Install dependencies:

	npm install

2. Start dev server:

	npm run dev

3. Build production files:

	npm run build

4. Preview production build:

	npm run preview

## Render Deployment

This project is configured for Render static deployment.

- Build Command: npm run build
- Publish Directory: dist

The included render.yaml also defines an SPA rewrite rule so client-side routes resolve to index.html.
